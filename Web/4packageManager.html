     새로운 패키지(프로그램)를 설치하려고 할 때 도와주는 소프트웨어 apt, yum 등이 있음.
     

     패키지 목록 업데이트
     $ apt-get update

     패키지 매니저를 통해 설치할 수 있는 패키지 목록들을 업데이트한다.
     (설치하기전에) 패키지 목록들이 업데이트

     패키지 찾기
     $ apt-cache search [패키지명]

     저장된 패키지 목록 중에 해당 패키지 찾기
     관련된 패키지 목록까지 나온다.

     패키지 설치
     $ apt-get install [패키지명]

     패키지 업그레이드
     $ apt-get upgrade
     $ apt-get upgrade [패키지명]

     패키지 삭제
     $ apt-get remove [패키지명]

     패키지를 설치하는 순서 (항상 이 순서를 따르는게 좋음)
     패키지 목록 업데이트 (apt-get update)
     패키지 설치 (apt-get install)
