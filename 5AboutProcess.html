     프로세스 목록 확인
     $ ps (shell 에서 실행중인 프로세스)
     $ ps -e (실행중인 모든 프로세스)
     * $ps -f (프로세스의 모든 상태 정보 확인)

     백그라운드 프로세스 목록까지 확인
     $ ps aux
     $ ps aux | grep apache2 : 'ps aux'의 출력에서 apache2가 포함된 행 출력

     프로세스 강제 종료
     $ kill [pid]

     그 외 프로세스 목록 확인 명령어
     $ top
     $ htop (top과 비슷하지만, 시각적인 면에서 더 좋음)





     *시그널

     시그널은 프로세스의 보내는 일종의 이벤트로 시그널에 대한 처리의 책임은 일반적으로
     프로세스에 있기 때문에 항상 같은 방식으로 처리가 이루어지지 않는다.
     현재 터미널 환경에서 입력할 수 있는 특수문자들에 대해서는 $ stty -e 로 확인



     *사용할 수 있는 모든 시그널은 -l 옵션으로 확인 가능

     ex) $ kill -l
         HUP INT QUIT ILL TRAP ABRT EMT FPE KILL BUS SEGV SYS PIPE ALRM TERM URG STOP TSTP CONT CHLD TTIN TTOU IO XCPU XFSZ VTALRM PROF WINCH INFO USR1 USR2
